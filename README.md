# 📖 # Blog-app <a name="about-project"></a>

**# Blog-app** is a simple and user-friendly web application for creating and managing blog posts. The app allows users to sign up, create posts, like posts, and leave comments.


## 🗝️ Key Features <a name="key-features"></a>

- User Authentication: Users can sign up and log in to the application to manage their blog posts and interact with other users.
- Create Posts: Registered users can create new blog posts, providing a title, content, and optional images.
- Like Posts: Users can like their favorite blog posts to show appreciation for the content.
- Comments: Users can leave comments on blog posts to engage in discussions.
- User Profiles: Each user has a profile page displaying their bio, profile picture, and a list of their authored blog posts.
- Post Statistics: The app tracks and displays the number of likes and comments for each blog post.
- Responsive Design: The application is designed to work seamlessly on different devices, including desktops, tablets, and mobile phones.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- [Install Ruby](https://www.ruby-lang.org/en/documentation/installation/)
- Install the [rails gem](https://rubygems.org/gems/rails)

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone git@github.com:rugwizangoga/Blog-app.git
```

### Usage

To run the project, execute the following commands:

```sh
  rails server
```
